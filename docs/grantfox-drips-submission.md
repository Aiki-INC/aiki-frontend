# Aiki GrantFox and Drips Wave Submission Notes

## Project Positioning

Aiki is an open-source Web3 education platform for course creation, learner dashboards, verifiable certificates, and learning rewards.

For Stellar ecosystem programs, position Aiki as an education infrastructure project preparing Stellar-powered payment flows and Soroban-based certificate verification.

## Recommended Repositories to Submit

Start with:

- `aiki-frontend`
- `aiki-smart-contract`

Do not submit empty or unfinished repositories until they contain clear code, setup instructions, and contribution-ready issues.

## Minimum Readiness Checklist

- [ ] README clearly explains the project.
- [ ] Local setup instructions work.
- [ ] `npm run lint` passes for frontend.
- [ ] `npm run type-check` passes for frontend.
- [ ] `npm run build` passes for frontend.
- [ ] `forge build` passes for smart contracts.
- [ ] `forge test` passes for smart contracts.
- [ ] CONTRIBUTING.md exists.
- [ ] Issue templates exist.
- [ ] 8-12 focused issues exist.
- [ ] Issues have labels and difficulty levels.
- [ ] Stellar/Soroban relevance is clear.
